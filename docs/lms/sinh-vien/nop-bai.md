# Nộp bài tập

Hướng dẫn sinh viên **nộp bài tập** trực tuyến trong khóa học trên **Canvas LMS**.

## Các bước nộp bài

**Bước 1:** Vào khóa học → chọn **bài tập (Assignment)** cần nộp.

**Bước 2:** Đọc kỹ **yêu cầu** và **hạn nộp**.

**Bước 3:** Nhấn **Thêm bài nộp (Add submission)**.

**Bước 4:** **Tải tệp lên** (kéo–thả hoặc chọn tệp) hoặc nhập nội dung trực tiếp.

**Bước 5:** Nhấn **Lưu** → **Nộp bài**.

<!-- ![Nộp bài tập](/images/lms/sv-submit.png) -->

::: warning Lưu ý về hạn nộp
- Sau **hạn nộp**, hệ thống có thể **không nhận** bài hoặc đánh dấu **nộp trễ**.
- Nộp sớm để tránh sự cố mạng vào phút chót.
- Kiểm tra trạng thái hiển thị **"Đã nộp để chấm điểm"** sau khi hoàn tất.
:::
