# Tài khoản HUTECH ID

**HUTECH ID** là tài khoản định danh duy nhất giúp bạn đăng nhập (SSO — đăng nhập một lần) vào tất cả hệ thống công nghệ thông tin của trường: email, LMS, cổng sinh viên, cổng đào tạo…

## HUTECH ID gồm những gì?

- **Tên đăng nhập:** thường là mã số sinh viên/giảng viên (ví dụ: `21010001`).
- **Mật khẩu:** do bạn tự đặt và bảo mật.
- **Email định danh:** `mssv@student.hutech.edu.vn` (sinh viên) hoặc `tên@hutech.edu.vn` (giảng viên/nhân viên).

::: tip Mẹo
Chỉ cần nhớ **một** tài khoản HUTECH ID là bạn có thể dùng cho mọi hệ thống. Khi đã đăng nhập một hệ thống, bạn sẽ tự động được đăng nhập các hệ thống còn lại.
:::

## Bắt đầu nhanh

1. Truy cập cổng đăng nhập **`https://sso.hutech.edu.vn`**.
2. Nhập **tên đăng nhập** và **mật khẩu** HUTECH ID.
3. Chọn hệ thống bạn muốn sử dụng.
